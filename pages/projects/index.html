Rendara3D

Rendara3D is a 3D engine for drawing objects on the screen, written in C++ and modern OpenGL.

You can download a demo here (requires Windows).

Features include

  • Loading of 3D models.
  • Diffuse, Normal, Specular textures.
  • Directional lights, point lights and flashlights.
  • Directional and Omnidirectional shadow maps.
  • Instanced rendering of models.

You can visit the github page of the project here.

         

Cells

Cells is a C++/Python software for simulating the collective motion of biological cells crawling on a 2D substrate.

The code has been designed to be flexible for testing out many different models.

Features include

  • Python C++ interface for easy scripting
  • Integration of langevin equations of motion with custom forces.
  • Cells can be composed of multiple disks (for now, maximum of 2).
  • Contraction/Expansion cycle of cell body can be simulated.
  • Binary format (HDF5) for saving files.

I have developed Cells for this scientific study.

The github page of the project can be found here.

         

Physgood

Physgood is a physics engine for games.

It features

  • Integration of the equations of motion for particles (...soon for rigid bodies).
  • Continuous time detection of sphere-sphere, sphere-edge, sphere-wall collisions.
  • Acceleration of collision detection using fixed space partitioning and hash mapping.
  • Collision response and rigid body failsafe.
